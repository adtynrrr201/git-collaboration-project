## [1.0.0] - 2024-XX-XX

### Added
- Initial project structure for Git collaboration demonstration
- Comprehensive contribution guidelines in `docs/guides/CONTRIBUTING.md`
- Pull request template in `docs/templates/PULL_REQUEST_TEMPLATE.md`
- Git collaboration helper script in `utils/git-helper.sh`
- Multiple documentation files in various formats (Indonesian and English reports)
- Example source code implementation in `src/`
- Organized directory structure with dedicated folders for docs, src, and utils
- README files in multiple directories for better documentation
- Text and markdown versions of reports for accessibility
- License file in src directory only

### Changed
- Restructured project from initial flat layout to organized hierarchy
- Moved documentation to appropriate subdirectories (`docs/guides`, `docs/reports`, `docs/templates`)
- Improved README with clear directory structure and navigation
- Enhanced documentation with better formatting and organization
- Added more comprehensive file listings to reflect actual project contents

- File organization inconsistencies resolved
